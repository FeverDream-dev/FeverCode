# FeverCode Roadmap

## Milestone 0: Foundation

- Compile Rust binary.
- Launch full-screen TUI placeholder.
- Workspace detection.
- Config load/init.
- Provider registry.
- Agent registry.
- Safety policy tests.

## Milestone 1: Useful MVP

- Chat input in TUI.
- OpenAI-compatible streaming client.
- Z.ai provider.
- Ollama local provider.
- Repo map.
- Plan-only mode.
- Patch proposal rendering.
- User approval queue.
- Apply patch inside workspace.
- `fever doctor` checks.

## Milestone 2: Competitive Agent

- Gemini CLI bridge.
- MCP stdio client.
- Multi-agent planning and review.
- Git checkpoint system.
- Test/lint/typecheck autodetection.
- Risk classifier for tool calls.
- Theme polish: Egyptian portal UI.

## Milestone 3: Vibe-Coder Power

- Spray mode with hard boundaries.
- Endless mode with budgets and checkpoints.
- Project templates.
- Agent marketplace format.
- Provider onboarding wizard.
- Install script using GitHub Releases.

## Milestone 4: Commercial Launch

- Signed releases.
- Update command.
- Telemetry opt-in only.
- Paid provider presets or cloud sync.
- Legal license finalized.
